# Task Manager App

A simple task management application built with React.

## Features

- Add new tasks
- Mark tasks as complete/incomplete
- Delete tasks
- Tasks are saved in your browser (local storage)
- See how many tasks remain

## How to Use

1. Type your task in the input field
2. Press "Add Task" or hit Enter
3. Click the checkbox to mark as complete
4. Click the Ã— button to delete a task

## Installation

1. Clone this repository
2. Run `npm install`
3. Run `npm start` to launch the app

## Technologies Used

- React
- JavaScript
- CSS
- LocalStorage for data persistence
